This is an instruction on running the lab1, linker, for OS course

1. Compile: change the directory to where source files and Makefile are, and type "make" in cmd to compile.
An executable named linker will be generated

2. Running: after compilation, type "linker $input" in the cmd where $input represents the directory of an input file.
The result will be printed to the console.
